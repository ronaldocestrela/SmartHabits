# SmartHabits — Frontend (Blazor WASM)

Primeira versão do cliente Blazor WebAssembly (target .NET 10) para a aplicação SmartHabits.

Funcionalidades iniciais:
- Login e Cadastro (mock)
- Edição de perfil
- CRUD de hábitos
- Definição de metas de hábito
- Dashboard com marcação diária

Desenvolvimento:
- Projeto: `src/SmartHabit.Client`
- Rodar localmente: `dotnet run --project src/SmartHabit.Client`
